# BLOG-BACKEND-SYSTEM

# Project Overview

This project implements a backend system for a blog platform using Node.js, Express.js, and SQLite. The system provides APIs for user registration, blog management, and commenting functionality.

# Features

* User Registration
* Blog Creation
* Blog Retrieval
* Comment Management
* SQLite Database Integration

# Technologies Used

* Node.js
* Express.js
* SQLite

# API Endpoints

* POST /register
* POST /blogs
* GET /blogs
* POST /comments

# Database

SQLite database stores users, blogs, and comments.

# Deliverable

A functional backend API connected to a database supporting authentication, blog creation, and commenting features.

# Conclusion

The project demonstrates backend development concepts including API creation, database integration, and data management for a blogging platform.
